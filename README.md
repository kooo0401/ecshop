# README

## Description
 ECサイト(electronic commerce site)
 ユーザー登録・複数カテゴリの商品購入・ショッピングカート・
 請求書プリント機能（PDF出力）を再現しています。

 ## Features

- ユーザー新規登録・ログイン機能
- stripeを用いた商品購入・決済処理
- 中間テーブルを利用したショッピングカート機能
- 商品名・説明文にヒットした文字列の検索機能
- 注文履歴・請求書プリント機能

## Requirement

- Python 3.8
- Django 3.0.2
- Docker 19.03.5
- Docker-compose 1.24.1
- Postgresql 12.0
- Gunicorn 20.0.4
- Nginx 1.17.4
